# Claire Richards Portfolio

정적 HTML/CSS 사이트입니다. GitHub Pages에 그대로 올릴 수 있습니다.

## 파일 구조

```txt
index.html      Home
info.html       INFO
works.html      WORKS
styles.css      모든 스타일
assets/images   사진 이미지
assets/icons    인스타그램/페이스북 아이콘 이미지
```

## 수정하기 쉬운 부분

- Home 문구 위치: `styles.css`에서 `#home-text` 검색
- Home 큰 제목: `#home-title` 검색
- 케이크 이미지: `#home-image` 검색
- 하단 큰 로고: `#footer-logo` 검색
- 인스타/페북 아이콘: `assets/icons/instagram.svg`, `assets/icons/facebook.svg` 교체
- 인스타/페북 링크: 각 HTML에서 `https://www.instagram.com/`, `https://www.facebook.com/` 부분 교체

## 폰트

- 영어 기본: Helvetica
- 로고 / footer-logo: Styalista
- 한글: Pretendard

Styalista는 폰트 파일을 포함하지 않았습니다. 직접 소유한 웹폰트 파일을 연결하거나, PC에 설치되어 있어야 정확히 보입니다.
